ClimaTek - Projet de validation de BTS

Supervision température / hygrométrie d'une voie d'aération de CTA (Centre de Traitement d'Air) :
Capteur + microcontrôleur + base de donnees PostgreSQL + interface Grafana.

Retour aux projets

Le projet est toujours en cours de réalisation pour l'obtention de mon BTS CIEL, je suis actuellement en phase de réflexion et de définition technique.

Architecture (vue simple)

  • Après le déménagement de l'entreprise dans un nouveau local, un CTA (Centre de Traitement d Air) a été installé.
  • Ce CTA est en circuit fermé, il n'est pas relié à l'extérieur, rendant le dispositif non optimal.

Architecture (vue simple)

  • Capteur : DFROBOT SHT31 (température + hygrométrie) à installer sur la zone à mesurer.
  • Microcontroleur : ESP32 programmé en C++, qui va recevoir les données du capteur et envoyer des messages à la base de données et aux volets motorisés.
  • Transmission : WiFi.
  • Base de données : PostgreSQL.
  • Supervision : Grafana.

Objectifs

  • Mesurer la température et l'hygrométrie en continu sur une voie de CTA.
  • Centraliser les données dans une base PostgreSQL pour avoir un historique.
  • Visualiser facilement les mesures (courbes, tendances) via Grafana.
  • Mettre en place des seuils / alertes pour detecter rapidement une anomalie.

Supervision

  • Dashboards Grafana : vues temps réel + historique sur plusieurs jours/semaines.
  • Indicateurs : min / max / moyenne, variations, détection de rupture de mesure.
  • Objectif : rendre le suivi simple pour l'exploitation.

Technologies et outils utilisés

ESP32 DFRobot SHT31 C++ Wi-Fi PostgreSQL Grafana Linux
